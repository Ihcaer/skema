# Skema System Documentation

**A modular application consisting of two distinct frontend services (Main Site and Admin Panel) and a core backend API, all orchestrated via Docker Compose configurations managed in root folder.**

This repository serves as the single source of truth for architecture, setup, API reference, and development guidelines for the entire project.
